# Быстрая упаковка расширения

## Сборка production версии

Запустите одну команду:

```bash
npm run package
```

Или пошагово:

```bash
npm run build:prod
node build-production.cjs
```

## Результат

После выполнения команды будет создана папка `dist/` с готовым расширением.

## Упаковка для распространения

1. Заархивируйте папку `dist/` в ZIP файл
2. Назовите файл, например: `LMS-MAI-Quiz-Solver-v2.0.0.zip`
3. Распространите ZIP файл

## Установка пользователями

Пользователи должны:

1. Распаковать ZIP файл
2. Открыть `chrome://extensions/` или `edge://extensions/`
3. Включить "Режим разработчика"
4. Нажать "Загрузить распакованное расширение"
5. Выбрать распакованную папку

## Что отключено в production версии

- ✅ Кнопка "Очистить все данные" - **скрыта**
- ✅ Кнопка "Удалить" для отдельных записей - **скрыта**
- ✅ Dev mode функции - **отключены**
- ✅ Удаление данных - **заблокировано**

Все функции удаления данных отключены для защиты пользовательских данных.
