# Бесплатные альтернативы для постоянной синхронизации

## Проблема:
ngrok бесплатный план не дает статический домен (URL меняется при перезапуске)

## Решения:

### Решение 1: Cloudflare Tunnel (РЕКОМЕНДУЕТСЯ) ⭐

**Преимущества:**
- ✅ Полностью бесплатно
- ✅ Статический домен (можно использовать свой или бесплатный от Cloudflare)
- ✅ Без ограничений
- ✅ HTTPS из коробки
- ✅ Надежнее ngrok

**Установка:**

1. Зарегистрируйтесь на https://www.cloudflare.com/ (бесплатно)

2. Установите cloudflared:
   - Скачайте: https://github.com/cloudflare/cloudflared/releases
   - Распакуйте `cloudflared.exe` в папку `server`

3. Создайте туннель:
   ```powershell
   cd f:\lms\server
   .\cloudflared.exe tunnel login
   .\cloudflared.exe tunnel create quiz-solver
   ```

4. Создайте конфигурацию `config.yml`:
   ```yaml
   tunnel: quiz-solver
   credentials-file: C:\Users\airccs\.cloudflared\тут-будет-id.json
   
   ingress:
     - hostname: quiz-solver.yourdomain.workers.dev
       service: http://localhost:3000
     - service: http_status:404
   ```

5. Запустите туннель:
   ```powershell
   .\cloudflared.exe tunnel run quiz-solver
   ```

6. Получите бесплатный домен от Cloudflare или используйте свой

### Решение 2: Railway.app (ОЧЕНЬ ПРОСТО) ⭐⭐

**Преимущества:**
- ✅ Полностью бесплатно (до 500 часов/месяц)
- ✅ Автоматическое развертывание
- ✅ Постоянный URL
- ✅ HTTPS автоматически
- ✅ Не зависит от вашего ПК

**Установка:**

1. Зарегистрируйтесь: https://railway.app/
2. Создайте новый проект
3. Подключите GitHub репозиторий (или загрузите код)
4. Railway автоматически определит Node.js и развернет
5. Получите постоянный URL (например: `quiz-solver.railway.app`)

**Нужно создать файл для Railway:**

Создайте `Procfile` в папке `server`:
```
web: node server.js
```

### Решение 3: Render.com (ПРОСТО)

**Преимущества:**
- ✅ Бесплатный тариф
- ✅ Постоянный URL
- ✅ HTTPS автоматически

**Установка:**

1. Зарегистрируйтесь: https://render.com/
2. Создайте Web Service
3. Подключите GitHub или загрузите код
4. Получите URL

### Решение 4: Автоматическое обновление URL в расширении

Если хотите использовать ngrok, можно создать скрипт, который автоматически обновляет URL:

1. ngrok предоставляет API для получения текущего URL
2. Расширение может периодически проверять и обновлять URL
3. Или использовать ngrok webhook для уведомления об изменении

## Рекомендация:

**Для постоянной работы:** Используйте **Railway** или **Render** - это проще и надежнее.

**Для тестирования:** Используйте **Cloudflare Tunnel** - бесплатно и стабильно.

**Для быстрого теста:** Используйте ngrok (но URL будет меняться)
