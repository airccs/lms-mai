# Развертывание на Railway (РЕКОМЕНДУЕТСЯ)

## Почему Railway?

- ✅ Полностью бесплатно (500 часов/месяц)
- ✅ Постоянный URL (не меняется)
- ✅ HTTPS автоматически
- ✅ Не зависит от вашего ПК
- ✅ Автоматическое развертывание
- ✅ Очень просто настроить

## Пошаговая инструкция:

### Шаг 1: Подготовка кода

1. Создайте файл `Procfile` в папке `server`:
   ```
   web: node server.js
   ```

2. Убедитесь, что в `package.json` есть скрипт `start`:
   ```json
   "scripts": {
     "start": "node server.js"
   }
   ```

### Шаг 2: Регистрация на Railway

1. Зайдите на https://railway.app/
2. Нажмите "Start a New Project"
3. Войдите через GitHub (или создайте аккаунт)

### Шаг 3: Создание проекта

**Вариант A: Через GitHub (рекомендуется)**

1. Создайте репозиторий на GitHub
2. Загрузите код в репозиторий
3. В Railway нажмите "New Project" → "Deploy from GitHub repo"
4. Выберите ваш репозиторий
5. Railway автоматически определит Node.js и развернет

**Вариант B: Прямая загрузка**

1. В Railway нажмите "New Project" → "Empty Project"
2. Нажмите на проект → "Settings"
3. Загрузите папку `server` как ZIP
4. Railway автоматически развернет

### Шаг 4: Настройка переменных окружения (опционально)

Если нужно изменить порт:
- В Railway: Settings → Variables
- Добавьте: `PORT=3000`

### Шаг 5: Получение URL

1. После развертывания Railway создаст URL
2. Нажмите на проект → "Settings" → "Domains"
3. Скопируйте URL (например: `quiz-solver.railway.app`)
4. Или создайте кастомный домен

### Шаг 6: Настройка расширения

В расширении укажите:
```
https://quiz-solver.railway.app
```

(Или ваш URL от Railway)

## Готово!

Теперь сервер работает 24/7, не зависит от вашего ПК, и URL не меняется!

## Обновление кода

Если нужно обновить код:
1. Обновите код в GitHub
2. Railway автоматически переразвернет
3. Или в Railway: Deployments → Redeploy
